# Haze Theme for Firefox
## Description
Firefox theme based on the [Haze](https://github.com/liferuin/haze) color palette.

## Installation
### Mozilla Add-ons
1. Go to [addons.mozilla.org/addon/haze](https://addons.mozilla.org/addon/haze/)
2. Click `Install Theme`.
